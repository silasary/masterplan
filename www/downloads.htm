Installing Masterplan

  • To install - Download the ZIP files for the latest version of Masterplan and its sample libraries, and unzip them into a new folder on your computer.
  • To upgrade - Download the latest version and extract the files into your Masterplan folder, overwriting the existing files. Your projects and libraries will work with the new version.

To run Masterplan you need to have .NET 3.5 installed on your computer (if you're using the downloadable WOTC Character Builder or Adventure Tools suite, you already have this; if not, you can download it here - like Masterplan, it's free).
